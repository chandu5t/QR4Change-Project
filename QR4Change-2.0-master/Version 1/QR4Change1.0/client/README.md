## 🌐 Starting the Frontend

1. Navigate to the **frontend** folder:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the frontend development server:
   ```bash
   npm run dev
   ```

The frontend will now be running, usually on **http://localhost:5173/** (if using Vite) or the port specified in your project.
